# 更新日志

本项目的所有重要更改都将记录在此文件中。

## [1.0.5] - 2026-03-07

### 🛠️ 优化
- **优化插件刮削流程**。

### ✨ 新功能
- **增加系列书籍，可以手动添加系列书籍**。

## [1.0.4] - 2026-03-06

### 🐛 修复与改进
- **修复MP3播放问题**：解决了在某些情况下，MP3文件无法正常播放的问题。

## [1.0.3] - 2026-03-04

### 🐛 修复与改进
- **适配最新后端**：跟进后端 API 数据结构变更，确保书籍元数据、章节列表及用户配置的正确解析与同步。
- **章节播放逻辑优化**：修复了存在番外章节时，正文/番外切换不稳定的问题。现在播放列表严格跟随当前选中的标签页（正文或番外），不再自动跳转。
- **章节编号修正**：配合后端更新，解决了正文章节编号因番外插入而不连续的问题。

## [1.0.2] - 2026-02-17

### ⚡️ 性能优化
- **离线体验**：优化离线登录后的缓存播放体验，修复了章节列表时长显示异常的问题。

## [1.0.1] - 2026-02-15

### ✨ 新增功能
- **离线能力增强**：实现了本地缓存管理，支持离线登录和播放已缓存的书籍。
- **界面优化**：优化了空状态下的界面显示，统一了移动端书架和收藏页面的视觉风格。

## [1.0.0] - 2026-02-09

### 新增功能
- Android 应用首次发布。
- 支持手动输入自托管服务器地址。
- 自动处理服务器地址重定向。
- 移动端优化的缓存管理器（LRU 策略）。
- 原生音频控制通知栏。
- 应用图标与启动画面。
